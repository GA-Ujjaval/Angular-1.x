DNI (Do Not Include) objects in BOM
Allow DNI objects in BOM
NOTE: When this setting is ENABLED, the system will allow more than one line item in the BOM to have the same Part Number, as long as the Reference Designators are different (eg: P100 (Ref Des = R1), P100 (Ref Des = R2)). This allows users to insert DNI (Do Not Include) or Non-Fitted parts in the BOM.

When DISABLED, the system will not allow users to have multiple line items with the same Part Number in a BOM.
