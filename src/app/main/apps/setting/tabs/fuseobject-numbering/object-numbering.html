Revision Scheme and Number Behavior
Allow edits to object numbers, major and minor revisions
Create FusePLM Part Numbers for line items (in file being imported) that do not have part numbers
NOTE: (1) When an item in file does not match a part in FusePLM (based on the 'Matching Criteria' defined by the user in the Import Manager),
       FusePLM will create a Part Number for it, even if the line item is missing a part number.
   (2) This setting is only supported for 'Part Import' and Flat-BOM Import, and NOT for Hierarchical-BOM Import.
Revision Scheme :

Major Revision :  Numeric Alphabetical
Use minor revision
Minor Revision :  Numeric Alphabetical
