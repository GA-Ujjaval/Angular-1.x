Edit Parts (Level 2 and below) in BOM
Allow Edit Parts (Level 2 and below) in BOM
NOTE: When DISABLED (default state), system will allow users to edit ONLY 'BOM Notes' attribute value for Parts that are at level 2 or below. Also, those parts cannot be replaced/removed from the BOM.
When ENABLED, system will allow users to edit any BOM-level attribute (Quantity, Reference Designator, Location, BOM Notes) for Parts that are at level 2 or below. Users can also replace/remove these parts.
