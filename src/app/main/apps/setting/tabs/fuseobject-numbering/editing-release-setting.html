Editing Released Objects
Allow edits to certain attributes
Note: Enabling this setting will allow users to edit 'Cost', 'Cost Type', 'Quantity on Hand' and 'Quantity on Order' attributes. When editing an object Manually, all 'Released' revisions of the object can be edited. However, when editing these attributes, through the 'Import > Update' process, only the current revision of any object('Released' or 'InDevelopment') will be updated
Allow edits to Bill of Materials
Note: Enabling this setting will allow users to perform edits within a Released Bill-of-Material (eg: Import Bill-of-Material, add part to BOM, delete part from BOM, modify RefDes, BOM Quantity, BOM notes). Also, when an object within a Released BOM is revised, users will be able to update that object's revision within a Released BOM.
Allow edits to the 'Attachments' tab
Note: Enabling this setting will allow users to upload attachment(s), associate document objects and remove them from a 'Released' object
Allow edits to the 'Sourcing' tab
Note: Enabling this setting will allow users to edit the 'Sourcing' tab in a 'Released' object (associate/remove manufacturing/sourcing objects)
Allow edits to 'Additional Info' tab
Note: Enabling this setting will allow users to edit custom attributes in a 'Released' object.
